# deepiMessage

A Python library for sending and receiving iMessages on macOS. 

## Usage

install the following dependencies:

```bash
pip install sqlite3
pip install typedstream
pip install python-dotenv
```

run main.py

```bash
python main.py
```

- Send iMessagesï¼š

- Receive iMessages
